### Install

```bash
curl -fsSL https://raw.githubusercontent.com/eulke/fnc/main/install.sh | bash
```

#### Source
If you use Bash, you can source the file in your `.bashrc` or `.bash_profile`:

```bash
source ~/.bashrc
```

if you use Zsh, you can source the file in your `.zshrc`:

```bash
source ~/.zshrc
```

### Usage

```bash
fnc
```
